# Quark-Gluon Jet Classification with GNNs

This project implements Graph Neural Networks to classify particle jets from the Quark-Gluon dataset.

## Setup
1. Clone the repository.
2. Install dependencies: `pip install -r requirements.txt`.
3. Place your `.npz` files in the `/data` folder.
4. Run training: `python main.py`.

## Models
- **GCN**: Graph Convolutional Network (Baseline).
- **GAT**: Graph Attention Network (High performance).
